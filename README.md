# TRADL

TRADL is a web app to find and trade stocks

## Usage

### Development mode

```shell
yarn # Install dependencies
cd packages/app
yarn run start # Run a watching development server
```

### Production mode

```shell
yarn # Install dependencies
cd packages/app
yarn run build # Compile for production
yarn run serve # Start the server
```
